What is ClickNeko?

ClickNeko is a clean, pastel-themed macro clicker built for gamers, learners, and creators. No bloat, no spyware — just cute, clean automation with open code you can learn from.

  • Toggle and hold-to-click modes
  • Click patterns and delays
  • Optional sound effects
  • Beginner-friendly, open Python source

Why It Matters

Most macro tools online are shady. ClickNeko is built around ethical coding and transparency. The full source code is available on GitHub — learn how it works and stay safe online.
